# Useful_Bash_Scripts
## Installation
Put the function you needed into ~/bin/!
## Functions
### 1. Swapfile
Usage:
```
  $echo 1 > 1.txt
  $echo 2 > 2.txt
  
  $swapfile 1.txt 2.txt         //Command to swap 1.txt and 2.txt
  Swapping 1.txt and 2.txt...
  Done
  
  $cat 1.txt
  2
  $cat 2.txt
  1
```
